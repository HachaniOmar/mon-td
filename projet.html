Notre Projet

Définition

Notre projet est une simulation de la bourse conçue pour permettre aux utilisateurs de comprendre et d'interagir avec les marchés financiers de manière interactive. Il s'agit d'une plateforme éducative et ludique qui reproduit les mécanismes réels de la bourse.

L'objectif principal est de fournir un environnement réaliste où les utilisateurs peuvent apprendre les bases de l'investissement, gérer un portefeuille, et réagir aux événements économiques.

Fonctionnalités

Le projet comprend plusieurs fonctionnalités clés pour simuler l'expérience de la bourse :

Choix du Mode de Difficulté

Les utilisateurs peuvent choisir entre différents niveaux de difficulté pour adapter la simulation à leur niveau d'expertise.

Création d'Entreprises

Créez des entreprises avec des caractéristiques spécifiques : capital de départ, secteur d'activité, taille, etc.

Suivi des Performances

Visualisez les performances des entreprises, y compris les bénéfices, les pertes, et les dividendes distribués.

Événements Aléatoires

Des événements aléatoires comme des crises économiques ou des innovations technologiques influencent le marché.

Achat et Vente

Achetez et vendez des actions et des obligations pour gérer votre portefeuille.

Gestion de Portefeuille

Suivez la performance de votre portefeuille en temps réel avec des indicateurs détaillés.

Les technologies

Pour réaliser ce projet, nous utilisons des technologies modernes et robustes, adaptées à la création d'une simulation boursière interactive. Voici les principaux outils et langages que nous avons choisis :

Java

Java est le langage principal de notre projet. Grâce à sa portabilité et sa robustesse, il nous permet de développer une application stable et performante pour la simulation boursière.

Java Swing

Java Swing est utilisé pour créer l'interface graphique de notre application. Il offre des composants intuitifs pour concevoir une interface utilisateur conviviale et réactive.

Événements en Temps Réel

Des événements en temps réel sont simulés pour reproduire les fluctuations du marché boursier. Cela inclut des crises économiques, des annonces de dividendes, etc.

Pourquoi Java et Java Swing ? Java est un langage polyvalent et largement utilisé dans le développement d'applications de gestion. Couplé à Java Swing, il permet de créer des interfaces graphiques riches et interactives, parfaites pour notre simulation boursière.

Exemple

Voici un exemple de scénario dans notre simulation :

Scénario : Une crise économique survient, affectant le secteur technologique. Les actions des entreprises technologiques chutent de 20 %. Les utilisateurs doivent décider s'ils vendent leurs actions ou attendre une reprise.

Résultat : Les utilisateurs qui ont vendu rapidement ont limité leurs pertes, tandis que ceux qui ont attendu ont vu leurs actions remonter après quelques mois.
